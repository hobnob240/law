# Law and Son Decorators Website

This version has been rewritten to sound like a real published business website rather than a website concept.

## Pages
- `index.html` — homepage
- `about.html` — about page
- `services.html` — services page
- `gallery.html` — gallery page
- `areas.html` — areas covered
- `enquiry.html` — quote form page

## Before you publish
1. Connect the form in `enquiry.html` to Formspree or another form handler
2. Replace the placeholder gallery images with real project photos
3. Update the canonical URLs if the final domain is different
4. Add a `CNAME` file if you are using a custom domain on GitHub Pages

## Publishing on GitHub Pages
Upload all files to your repo, then publish from the root of the `main` branch in Settings → Pages.
